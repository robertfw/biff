## Contributing

To hack on Biff:

1. `cd example`
2. `cp config.edn.TEMPLATE config.edn`
3. `cp config.sh.TEMPLATE config.sh`
4. `./task dev`

(The example project's deps.edn declares a local dependency on the Biff library
code.)
